Contact Management Web (MERN Stack)

A simple Contact Management Web Application built using the MERN stack to demonstrate full-stack development fundamentals, REST API usage, and clean UI with Tailwind CSS.

Tech Stack

Frontend

-React.js
-Tailwind CSS
-Axios
-useState / useEffect

Backend

-Node.js
-Express.js
-MongoDB
-Mongoose

Features

Add new contacts using a form
Client-side form validation
Store contacts in MongoDB
Fetch and display contacts without page reload
Delete/Edit contact
Basic sorting by latest contacts
Responsive UI using Tailwind CSS
